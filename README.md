\# Healthcare Data Analysis Project



\## Project Overview



This project performs an \*\*end-to-end analysis of healthcare data\*\* to uncover insights related to patient demographics, hospital admissions, treatment costs, and length of hospital stay.



The project combines \*\*Python for data processing and exploratory data analysis, MySQL for querying structured data, and Power BI for interactive dashboards\*\* to deliver meaningful insights that can support healthcare decision-making.



---



\# Project Objectives



\* Analyze patient demographics such as \*\*age groups and gender distribution\*\*.

\* Understand patterns in \*\*hospital admissions and length of stay\*\*.

\* Identify trends in \*\*medical conditions and treatment costs\*\*.

\* Evaluate \*\*hospital and doctor performance based on billing amounts\*\*.

\* Create \*\*interactive dashboards\*\* to visualize healthcare insights.

\# Project Workflow



\### Data Loading



The dataset was loaded using \*\*Pandas\*\* for further analysis.



\### Data Cleaning



Several preprocessing steps were performed:



\* Removed \*\*duplicate records\*\*

\* Handled \*\*invalid billing values\*\*

\* Converted \*\*column names to lowercase\*\*

\* Created new features such as:



&nbsp; \* \*\*Length of Stay\*\*

&nbsp; \* \*\*Stay Category\*\*

&nbsp; \* \*\*Age Category\*\*



\### Exploratory Data Analysis (EDA)



EDA was performed to understand patterns and relationships in the dataset, including:



\* Patient distribution by age and gender

\* Admission types

\* Hospital stay duration

\* Billing patterns across hospitals and doctors



\### SQL Analysis



The cleaned dataset was analyzed using \*\*MySQL queries\*\* to extract meaningful insights and perform structured data analysis.



\### Data Visualization



Visualizations were created using \*\*Matplotlib and Seaborn\*\* to highlight key patterns in the data.



\### Dashboard Creation



An \*\*interactive Power BI dashboard\*\* was developed to present the insights visually and allow users to explore healthcare metrics.



---



\# Key Insights



\* \*\*Adults and seniors account for the majority of hospital admissions.\*\*

\* \*\*Extended hospital stays are common\*\*, indicating higher demand for long-term care.

\* Certain \*\*doctors and hospitals contribute significantly to total billing revenue.\*\*

\* Healthcare costs vary across \*\*medical conditions and insurance providers\*\*.

\* Admission type influences \*\*length of hospital stay\*\*.
